# AI Chatbot - Powered by Claude

> A clean, minimal AI chatbot built as a single HTML file. No backend, no install, no build step.

## Live Demo

https://nzjulien.github.io/ai-chatbot

## Features

- Real-time chat with Claude AI
- Full conversation history (multi-turn)
- API key stored locally in browser only
- Typing indicator with animated dots
- Auto-resize textarea
- One-click suggestions to get started
- Clear chat button
- Mobile friendly

## Setup

1. Open the live demo link above
2. Enter your Anthropic API key (get one at console.anthropic.com)
3. Start chatting

## Stack

- Vanilla HTML/CSS/JS - zero dependencies
- Anthropic Claude API (claude-sonnet-4-6)
- localStorage for key persistence
